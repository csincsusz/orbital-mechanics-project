# Orbital Mechanics Project

Ez egy hosszú távú tanulási projekt, amelynek célja, hogy alapszintű Euler-integrálástól eljussak egy alapszintű orbitális mechanika / küldetéstervező szimulátorig.

## Célok

- Egy nagy, végigvitt projekt elkészítése elejétől a végéig
- A numerikus szimuláció és a mögötte lévő matematika/fizika megértése
- A git és GitHub verziókezelés gyakorlati elsajátítása
